# Senier_Project

## Unit

1. 이미지 Rekognition CLI 환경에서 테스팅 완료 -> 버튼을 구현 안함
2. 이미지 업로드 테스팅 완료
3. 오디오 업로드 테스팅 완료
4. ETRI API 오디오 테스팅 완료
5. 이미지 올린 후 Rekognition 을 통해 얻은 값 index 에 보이기 
6. 업로드할 이미지의 프리뷰 보이기

## 해야할 일

1. 오디오가 저장되는데 저장 디렉토리를 변경시켜야 한다

2. rekognition 으로 얻은 값들을 horizontal scroll 을 통해 출력하고 싶다
